# Game Changers List — Commander Brackets

Última actualización: Octubre 21, 2025 (CFP Update)

## Qué son los Game Changers

Cartas identificadas por el Commander Format Panel como capaces de distorsionar
significativamente una partida de Commander. No están baneadas, pero están
restringidas por bracket:

- **Brackets 1-2**: No permitidos
- **Bracket 3**: Algunos pueden estar presentes, depende del contexto
- **Brackets 4-5**: Esperados y aceptados
